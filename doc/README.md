# Documentation

PDF guides for the solar-powered glass tumbler.

| File | Description |
| :--- | :--- |
| [`detailed_guide.pdf`](detailed_guide.pdf) | Comprehensive manual covering the full system description, troubleshooting, electrical schematics, and repair procedures. |
| [`quickstart_guide.pdf`](quickstart_guide.pdf) | Short operator-facing guide for daily loading, running, and unloading of the tumbler. |

The rendered HTML version of the user-facing content lives on the project's [GitHub Pages site](https://global-health-engineering.github.io/glass-tumbler/).
