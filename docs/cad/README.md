# CAD

CAD resources for the solar-powered glass tumbler.

## Subdirectories

- [`drawings/`](drawings/) — 2D technical drawings in PDF form, for manufacturing reference.
- [`step/`](step/) — 3D CAD assemblies exported in the neutral STEP (ISO 10303) format, openable in any major CAD package (SolidWorks, Fusion 360, FreeCAD, Onshape, etc.).

## Notes

- All dimensions are in millimetres unless annotated otherwise.
- The support wheel on the drum plate (added during field testing) and the outer-frame reinforcement beams are **not** present in these CAD files, as they were later modifications. See the construction guide for details.
