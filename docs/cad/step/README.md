# STEP Files

3D CAD assemblies in the neutral STEP (ISO 10303) format. These can be opened in any modern CAD package (SolidWorks, Fusion 360, FreeCAD, Onshape, etc.).

| File | Contents |
| :--- | :--- |
| `Tumbler Assembly.step` | Full tumbler assembly (outer frame + inner frame + drum + drivetrain mount). |
| `Main Frame.step` | Outer (main) frame sub-assembly. |
| `Drum Frame.step` | Inner (drum) frame sub-assembly with bearing backplate. |

## Not represented in the CAD model

- The auxiliary support wheel added to the drum plate after field testing.
- The outer-frame reinforcement and crossbeams added after initial welding.
- The lockable motor/drivetrain housing built in Malawi.

See `construction.qmd` (or the detailed guide PDF) for these additions.
