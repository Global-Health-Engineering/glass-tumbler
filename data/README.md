# Data

Datasets supporting the performance, efficiency, and financial analyses reported in [`results.qmd`](../results.qmd).

## Structure

- [`raw_data/`](raw_data/) — measurements collected directly from field trials and external sources, as originally recorded.
- [`derived_data/`](derived_data/) — processed datasets produced by analysis scripts from the raw inputs.
- [`metadata/`](metadata/) — data dictionary, file inventory, and collection/methodology documentation.

## Licensing

## Start here

Open [`metadata/README.md`](metadata/README.md) for the full dataset description, collection dates and location, methodology, and instrument/software information. See [`metadata/codebook.csv`](metadata/codebook.csv) for variable-level definitions and units, and [`metadata/file_list.csv`](metadata/file_list.csv) for a machine-readable file inventory.
